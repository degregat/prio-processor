# SWIG-Wrapper Example Usage

This example demonstrates usage of the generated wrapper around the libprio
functions. The wrapper is no longer maintained within this repository and has
moved to [the python wrapper of
libprio](https://github.com/mozilla/libprio/tree/master/python). It is used
heavily within the prio-processor, however.

## Running the example

With docker, run from the current directory.

```bash
docker run -v $(pwd):/app -it prio:dev python3 main.py
```
